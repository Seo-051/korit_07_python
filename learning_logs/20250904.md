## 🧩 1. Hangman 게임 예제 분석

Python으로 구현한 **행맨(Hangman)** 게임을 통해 **리스트, 반복문, 조건문, 문자열 처리**, 그리고 특히 **함수 정의/호출**에 대해 실습한 코드입니다.

### 📌 핵심 기능

- 정답 단어 리스트에서 하나를 무작위로 선택
- 사용자가 알파벳을 입력하여 정답을 추측
- 틀릴 경우 목숨(lives)이 감소하며, 그에 따라 hangman 그림이 완성됨
- `display` 리스트를 통해 단어 추측 상태를 시각적으로 표현

### 📎 코드 주요 포인트

```python
import random

# 단계별 그림
stages = ['''...''', '''...''', '''...''']  # 생략

# TODO 1: 목숨 설정
lives = 6

# TODO 2: 정답 단어 설정 및 display 초기화
word_list = ['apple', 'banana', 'camel']
chosen_word = random.choice(word_list)
display = ['_' for _ in chosen_word]

# TODO 3: 게임 루프
end_of_game = False
while not end_of_game:
    print(stages[lives])
    guess = input('알파벳을 입력하세요 >>> ').lower()

    for i in range(len(chosen_word)):
        if chosen_word[i] == guess:
            display[i] = guess

    # 틀린 경우 처리
    if guess not in chosen_word:
        lives -= 1
        print(f'당신의 기회는 {lives} 번 남았습니다.')
        if lives == 0:
            print('모든 기회를 잃었습니다.')
            end_of_game = True
            print(f'정답은 {chosen_word}입니다.')

    # 정답 맞췄는지 확인
    if '_' not in display:
        print('정답입니다 !! 💌')
        end_of_game = True
        print(f'정답은 {chosen_word}입니다.')

    # 현재 상태 출력
    print(' '.join(display))

📘 함수의 종류
유형	설명	예시

📌 함수 용어
함수 정의: def 키워드로 작성

인수 (argument): 함수에 전달하는 입력값

매개변수 (parameter): 인수를 받는 변수

리턴값 (return): 함수의 출력

함수 호출 (call): 함수를 사용하는 행위

🧪 함수 리턴값에 따른 4가지 유형
